khởi tạo backend Express + MySql
    npm init -y
    npm i express sequelize mysql2 dotenv cors morgan bcryptjs jsonwebtoken
    npm i -D nodemon
khởi tạo cấu trúc thư mục
    backend/
    │── index.js            (file chính)
    │── db.js               (kết nối MySQL)
    │── .env                (thông tin server MySQL)
    │── routes/             (các route API)
    │── controllers/        (xử lý logic)
    │── middleWares/        (xác thực)
